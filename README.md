# DTM Worklog Prototype
# DTM Worklog

DTMer / ボカロP 向けの、作業時間を記録してあとから振り返るための Web アプリです。  
「制作物ごと」「工程ごと」に時間を積み上げて、どの工程にどれくらい時間がかかったかを可視化できます。

このアプリは **ブラウザ内にローカル保存** されることを前提にしており、個人利用や小規模な共有向けに軽く使える構成を目指しています。

---

## 主な機能

- 制作物単位でタスクを管理
- 工程単位で作業開始 / 終了
- 並列作業に対応
- 任意メモ付きでログ記録
- 分析画面で
  - 制作物ごとの時間
  - 作業項目ごとの時間
  - ログ一覧
  - 各制作物の工程内訳
  を確認可能
- タスク編集 / ログ編集 / ログ削除
- タスクの凍結 / 完了 / アーカイブ
- アーカイブ済みタスクの復元
- IndexedDB 保存
- JSON バックアップ / 復元
- CSV 出力
- 全データ削除
  - 確認ダイアログで `削除` と入力した場合のみ実行

---

## 想定ユーザー

- DTM をする人
- ボカロP
- 制作の作業時間をあとから振り返りたい人
- 日々の入力負荷をできるだけ小さくしたい人

---

## 作業フロー

### 開始時
1. 制作物を選ぶ
2. 作業項目を選ぶ
3. 必要ならメモを書く
4. 開始する

### 終了時
1. 終了したい制作物を選ぶ
2. 必要なら終了メモを書く
3. 終了する

---

## 標準作業項目

標準では以下の順で表示されます。

1. 作曲
2. 編曲
3. 作詞
4. 調声
5. 修正
6. ミックス
7. マスタリング
8. その他

作業項目は追加可能です。  
将来的に標準項目の途中へ新しい項目を差し込みやすいよう、並び順は `sortOrder` ベースで管理する想定です。

---

## 保存について

このアプリのデータは **このブラウザ内** に保存されます。

- 別ブラウザには引き継がれません
- 別 PC にも自動では引き継がれません
- ブラウザデータを消すと失われる可能性があります

そのため、定期的な **JSON バックアップ** を推奨します。

---

## バックアップ / 復元

### JSON バックアップ
アプリ内のデータを JSON として書き出します。

### JSON 復元
バックアップ済み JSON を読み込み、現在のデータを丸ごと置き換えます。  
復元前に、現在のデータを先にバックアップしておくことをおすすめします。

### CSV 出力
ログ一覧を CSV で書き出します。  
スプレッドシートで確認したいときに使えます。
